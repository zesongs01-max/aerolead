Interactive Endpoint Explorer

POST /v1/search/people

Query contact database using faceted filters.

curl -X POST http://127.0.0.1:8000/v1/search/people \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "marketing ops",
    "filters": {
      "company_locations": ["United States"],
      "seniorities": ["director"]
    }
  }'
POST /v1/enrich/person

Run contact through enrichment waterfall.

curl -X POST http://127.0.0.1:8000/v1/enrich/person \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": {
      "full_name": "Jane Doe",
      "company_domain": "acme.com"
    }
  }'
